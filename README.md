# LessonsSpring
Practicing all the features of Spring
essonsSpring
This project is a training project for practicing the features of Spring framework. Spring framework is a comprehensive programming and configuration model for modern Java-based enterprise applications. Spring framework supports various application scenarios, such as web, data access, messaging, testing, and integration.

Features
The project consists of several modules that demonstrate different aspects of Spring framework. The modules are:

spring-core: This module shows how to use dependency injection, events, resources, i18n, validation, data binding, type conversion, and SpEL with Spring core container.
spring-aop: This module shows how to use aspect-oriented programming with Spring AOP module.
spring-data: This module shows how to use transactions, DAO support, JDBC, ORM, and marshalling XML with Spring Data Access module.
spring-web: This module shows how to use Spring MVC and Spring WebFlux web frameworks with Spring Web module.
spring-boot: This module shows how to use Spring Boot to create stand-alone applications that can run with embedded servers or as executable jars.
